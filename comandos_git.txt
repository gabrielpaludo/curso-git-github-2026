git init . 	      		-> cria um repo git na pasta atual
git status 			-> mostra o status do trabalho
git add nome_arquivo   	 	-> adiciona arquivo para ser commitado
git commit -m "msg" 		-> faz commit do que estava para ser commitado
git log 			-> mostra todo o histórico de commit
git checkout -b nome_da_branch	-> cria uma branch
git checkout nome_da_branch	-> vai para a branch
git branch			-> mostra a lista de branch
git branch -d nome_da_branch	-> deleta a branch
git merge nome_da_branch	-> mescla uma branch com a outra (normalmente main)
 
## FLUXO DE TRABALHO

1. criar o diretório do projeto
2. iniciar o git dentro do diretório criado
3. edita ou cria arquivos
4. git status
5. git add <nome_arquivos>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3
